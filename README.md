# Dream Nest Aviary & Farm Management System

A comprehensive farm management web application for pigeon and chicken breeders.

## Features

- 🔐 Authentication with NextAuth.js (Email/Password + Google)
- 📊 Admin Dashboard with analytics and charts
- 🏗️ Project Management
- 🐦 Pair/Bird Management
- 🥚 Breeding Records Tracking
- 💰 Expense Management with CSV export
- 🖼️ Photo Gallery with public access
- 📈 Reports and Analytics

## Tech Stack

- **Framework:** Next.js 15 with App Router
- **Language:** TypeScript
- **Database:** MongoDB with Mongoose
- **Authentication:** NextAuth.js
- **UI:** shadcn/ui + Tailwind CSS
- **Forms:** React Hook Form + Zod
- **Charts:** Recharts
